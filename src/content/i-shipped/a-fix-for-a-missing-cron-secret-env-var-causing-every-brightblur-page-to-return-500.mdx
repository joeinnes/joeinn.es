Brightblur's production site was returning HTTP 500 on every single request, including the homepage. The root cause was a `validateEnv` function that ran on every request and threw an error when CRON_SECRET was missing from the environment. I fixed the validation so a missing non-critical variable doesn't crash the entire site.
